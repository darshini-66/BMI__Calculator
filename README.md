# BMI Calculator

A simple and responsive web application that calculates Body Mass Index (BMI) based on a user's height and weight inputs.

## Features

- BMI Calculation
- User-Friendly Interface
- Instant Results
- BMI Category Classification
- Responsive Design

## BMI Categories

- Underweight
- Normal Weight
- Overweight
- Obese

## Tech Stack

- HTML5
- CSS3
- JavaScript

## How It Works

1. Enter height and weight.
2. Click the Calculate button.
3. The application computes BMI using the standard formula.
4. Displays BMI value along with the corresponding health category.

## Formula Used

BMI = Weight (kg) / Height² (m²)
